## 目录结构

```
video/
├── input/          # 源视频文件
├── output/
│   ├── srt/        # 字幕文件（.srt）
│   └── *.mp4       # 最终视频（_final.mp4）
├── process.sh      # 一键处理脚本
├── transcribe.py   # AI 语音识别（mlx-whisper）
└── 流程.md
```

## 使用方式

1. 把源视频放入 `input/` 目录
2. 运行脚本：

```bash
# 处理单个视频
./process.sh SVID_20260210_210732_1.mp4

# 处理 input 下所有视频
./process.sh --all
```

3. 在 `output/` 目录获取结果：
   - `xxx_final.mp4` — 静态封面 + 烧录字幕的压缩视频
   - `srt/xxx.srt` — 简体中文字幕文件

## 处理流程（脚本自动执行）

1. 提取第一帧作为封面
2. 静态封面 + 原始音频合成视频（大幅压缩体积）
3. AI 语音识别生成简体中文字幕（mlx-whisper + 繁转简）
4. 烧录字幕到视频（左上方显示）
5. 清理中间文件，只保留最终视频和字幕
